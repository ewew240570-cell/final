Git

  • git init - расшифровывается как "Initializing the repository", что переводится как "инициализация репозитория".
  • git status - расшифровывается как "Change status", что переводится как "статус изменений" или "проверить статус".
  • git add - расшифровывается как "Add files", что переводится как "добавить файлы".
  • git commit - расшифровывается как "Commit changes", что переводится как "зафиксировать изменения".
  • git push - расшифровывается как "Send to server", что переводится как "отправить на сервер" С помощью нее можно переместить, загрузить файлы.
